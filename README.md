Dicionários e Conjuntos em Python 🐍

Este repositório compila exemplos práticos sobre duas estruturas de dados essenciais em Python: **Dicionários** (`dict`) e **Conjuntos** (`set`). O material serve como apoio aos alunos para compreenderem como organizar e manipular coleções de dados de forma eficiente.

## 📚 O que você vai aprender

Nos arquivos contidos neste repositório, você encontrará exemplos práticos sobre:

* **`dict.py` (Dicionários):**
    * Criação de estruturas de mapeamento (chave: valor).
    * Acesso a dados e uso de métodos nativos como `.keys()`, `.values()` e `.items()`.
    * Iteração eficiente sobre o conteúdo do dicionário.

* **`set.py` (Conjuntos):**
    * Definição e uso de coleções de itens únicos (sem duplicatas).
    * Aplicação de operações fundamentais da teoria dos conjuntos, como união (`|`), interseção (`&`) e diferença (`-`).
    * Técnicas para remoção rápida de itens repetidos em listas.
